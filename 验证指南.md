## 测试流程

### 步骤 1：客户注册 → 登录

1. 打开 http://localhost:3000
2. 点击「注册」链接，填写用户名、密码、手机号（11 位手机号）完成注册
3. 注册成功跳转登录页，选择「客户登录」，用刚注册的账号登录

### 步骤 2：客户 — 引导查询向导

1. 登录后点击左侧「智能选房」（向导入口）
2. 按步骤依次：选择位置 → 选择户型 → 设置预算 → 查看结果
3. 验证最后一步展示匹配的楼盘列表

### 步骤 3：客户 — 自定义多条件查询

1. 点击左侧「房产查询」
2. 分别测试：关键字搜索（输入"深圳"）、位置模糊搜索、户型筛选、价格区间
3. 验证结果列表和分页正常
4. **刷新页面，验证搜索条件是否保留在 URL 中**

### 步骤 4：客户 — 查看房产商列表

1. 点击左侧「房产商列表」
2. 验证展示所有开发商的卡片列表
3. 点击某卡片查看详情

### 步骤 5：客户 — 提交建议

1. 点击左侧「提交建议」
2. 选择开发商、填写期望户型、可接受价格
3. 提交后验证成功提示

### 步骤 6：客户 — 个人中心

1. 点击左侧「个人中心」
2. 编辑个人信息（修改联系方式、添加购房意向标签）
3. 查看「我的建议」列表

### 步骤 7：客户 — 打印报表 / 下载 PDF

1. 在「房产查询」页搜索后，点击「打印报表」或「下载 PDF」
2. 验证浏览器打印对话框弹出 / PDF 文件成功下载

### 步骤 8：开发商 — 房产 CRUD

1. 退出客户，用 `bgy_admin / 123456` 以「开发商登录」登录
2. 点击「房产管理」，查看已有楼盘列表
3. 点击「添加房产」创建新楼盘，验证成功后列表刷新
4. 点击编辑修改楼盘信息，验证保存成功
5. 删除一条记录，验证确认对话框和删除成功

### 步骤 9：开发商 — 关联分析

1. 点击「空置率分析」（关联分析）
2. 验证三个图表正常渲染（按区域柱状图、按户型柱状图、楼层散点图）
3. 点击「导出 PNG」验证图片下载

### 步骤 10：开发商 — 公司信息

1. 点击「公司信息」或个人中心
2. 编辑公司名称、联系方式等，保存验证

### 步骤 11：权限验证

- 客户登录后尝试访问 `/developer/properties` → 应被重定向
- 开发商登录后尝试访问 `/customer/wizard` → 应被重定向

# 启动前端（新终端）
cd frontend
npm run dev
