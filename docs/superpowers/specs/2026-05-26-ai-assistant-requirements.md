# AI 购房助手 — 需求记录

> 日期：2026-05-26 | 版本：v1.1 | 类型：需求记录（非规格说明书）

## 核心架构

| 维度 | 选择 |
|------|------|
| LLM API | DeepSeek API（chat + embedding） |
| Java AI 框架 | Spring AI |
| 向量检索 | Apache Lucene 混合检索（BM25 + 向量） |
| 文档切分 | 固定 300 字一段 |
| 文档解析 | Apache Tika（PDF/Word） |
| 文件存储 | 服务器本地文件系统 |
| 前后端通信 | HTTP POST，JSON 响应（不流式） |
| 聊天历史 | 仅带最近 3 轮 |
| 索引策略 | 增量索引，手动上传文档驱动 |

## 功能清单

| 维度 | 选择 |
|------|------|
| 界面形式 | 右下角悬浮气泡/小窗，固定置顶，所有登录后页面可见 |
| 对话管理 | 多会话，默认空白新对话，支持创建/切换/删除 |
| 输出方式 | 一次性返回完整回答，不流式 |
| 输入方式 | 纯文本 |
| AI 人格 | 亲切友好的购房助手，口语化 |
| 核心能力 | 问答 + RAG 推荐 + 收藏（Function Calling） |
| 推荐逻辑 | 对话理解 + 用户画像综合，RAG 驱动 |
| 推荐卡片 | 名称 + 价格 + 区域 + 户型 + 缩略图，卡片样式；卡片上无按钮，用户说"帮我收藏"可触发 Function Calling 收藏；点击卡片跳转详情页 |
| 收藏功能 | 详情页手动按钮收藏 + AI Function Calling 收藏；Dashboard 底部展示收藏列表 |
| 知识库组成 | 平台业务数据 + 房地产领域知识 + 开发商自定义文档 |
| 开发商知识库 | 所有开发商共享知识库（不做隔离）；新增知识库管理页面（上传 PDF/Word / 删除 / 预览解析结果） |
| 失败处理 | 告知用户 AI 暂不可用 |
| 兜底策略 | 无法回答时友好告知 |
| 与 Wizard 关系 | 独立共存，互不影响 |
