# 登录页面 UI 改造设计

## 概述

将当前居中卡片布局的登录页面改造为左右分栏布局，采用温暖住宅风格配色，提升品牌视觉和用户体验。

## 布局结构

```
┌──────────────────────────────────────────────────┐
│  ┌──────────────────────┐  ┌──────────────────┐  │
│  │                      │  │                  │  │
│  │  品牌区域 (60%)       │  │  登录卡片 (40%)   │  │
│  │                      │  │                  │  │
│  │  暖橙→米咖渐变        │  │  暖白背景卡片      │  │
│  │  几何装饰圆形 + 弧线   │  │  表单内容          │  │
│  │  Logo + 品牌名 + 标语  │  │                  │  │
│  │                      │  │                  │  │
│  └──────────────────────┘  └──────────────────┘  │
└──────────────────────────────────────────────────┘
```

- 满屏高度（`100vh`），左右两栏
- 无内边距，无缝拼接

## 左侧品牌区

### 背景渐变
- 暖橙 → 米咖渐变（从左上到右下）
- 起始色: `#f5a623` → 终止色: `#d4a373`

### 几何装饰
- 3-4 个半透明大圆形散落（`border-radius: 50%`，白色 `rgba(255,255,255,0.12~0.18)`）
- 1-2 条柔和弧线穿越圆形，增加层次感
- 尺寸: 300px ~ 600px 不等，位于不同位置

### 品牌内容
- Logo: "房"字在暖色方块中（方块使用白色半透明或深褐色背景）
- 品牌名: "房易查" 白色或深褐色大字
- 标语: "让购房更简单" 深褐色小字
- 整体居中或左下对齐

## 右侧登录卡片

### 卡片样式
- 暖白背景: `#fdf8f3`
- 圆角: `16px`
- 阴影: `0 8px 32px rgba(180, 130, 80, 0.12)`
- 内边距: `40px`
- 宽度: `400px`，垂直居中

### 表单内容（从上到下）

| 元素 | 说明 |
|------|------|
| 标题 | "欢迎回来" 24px 粗体 + "请登录您的账号" 14px，色值 `#4a3728` / `#8a7a6a` |
| 角色切换 | 两个圆角按钮（"客户登录"/"开发商登录"），选中暖橙 `#f5a623` |
| 用户名 | 圆角 8px，边框 `#e8ddd0`，聚焦暖橙边框，前缀图标 User |
| 密码 | 同上，前缀图标 Lock |
| 登录按钮 | 全宽，暖橙填充 `#f5a623`，白色文字，悬停 `#e0961a` |
| 注册链接 | "还没有账号？立即注册"，暖橙色链接 |

### 配色表

| 用途 | 色值 |
|------|------|
| 卡片背景 | `#fdf8f3` |
| 主色调/按钮填充 | `#f5a623` |
| 按钮悬停 | `#e0961a` |
| 文字主色 | `#4a3728` |
| 文字次要 | `#8a7a6a` |
| 边框/分割线 | `#e8ddd0` |

## 实现范围

- 仅修改 `frontend/src/views/login/LoginView.vue`
- 保留所有逻辑不变（登录、角色切换、表单验证、注册跳转）
- 更新作用域样式，不影响其他页面
